## 食用方式

- 先把要下载的小说在知乎里加入书架
- 这个项目没有做架构，代码是屎山（以后可能会优化）
- 在my_file/下填入jsdom_path.txt和cookie.txt
- 修改ysg.py里的文件夹位置（默认是download）
- 依次运行 build.py - json2csv.py - clean.py - ysg.py
